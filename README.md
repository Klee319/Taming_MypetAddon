# MyPetAddon

MyPetプラグインを大幅に拡張するPaper APIアドオンプラグイン。
LevelledMobs連携レアリティ、性格、親密度、進化、固有スキル、装備、図鑑システムを追加します。

## 動作要件

| 項目 | バージョン |
|------|-----------|
| Minecraft | 1.21.x |
| サーバー | Paper 1.21.4+ |
| Java | 21+ |
| MyPet | 3.14.x（必須） |
| LevelledMobs | 4.x（任意） |
| MythicMobs | 5.x（任意） |

## インストール

1. `MyPetAddon-1.0.0-SNAPSHOT.jar` を `plugins/` フォルダに配置
2. サーバーを起動（初回起動時に `config.yml` が自動生成されます）
3. `config.yml` を編集してバランス調整
4. `/petadmin reload` で反映

## 機能一覧

### 1. テイミングシステム

**動物のテイム**
- 金のリンゴ（設定変更可）で右クリック → 即座にMyPetとして仲間に

**モンスターのテイム**
- ネザースター（設定変更可）で右クリック → そのモブを倒す → ダミーエンティティ出現
- `/petname <名前>` で命名して仲間に（30秒タイムアウト）

**除外設定**
- MythicMobsのカスタムMobは自動除外
- `config.yml` で任意のMobタイプを除外リストに追加可能

### 2. レアリティシステム

テイム時にレアリティが決定されます。LevelledMobs導入時はMobレベルに基づいて確率が変動します。

| レアリティ | 色 | ステータス倍率 | スキル枠 | 覚醒 |
|-----------|-----|---------------|---------|------|
| Common | 白 | x1.0 | 0 | - |
| Uncommon | 緑 | x1.2 | 0 | - |
| Rare | 青 | x1.5 | 0 | - |
| Epic | 紫 | x2.0 | 1 | - |
| Legendary | 金 | x3.0 | 2 | 可 |
| Mythic | 赤 | x4.0 | 3 | 可 |

**LevelledMobs連携**

| Mobレベル | Common | Uncommon | Rare | Epic | Legendary | Mythic |
|-----------|--------|----------|------|------|-----------|--------|
| 1-10 | 60% | 30% | 10% | - | - | - |
| 11-25 | 30% | 40% | 20% | 10% | - | - |
| 26-50 | - | 20% | 40% | 30% | 10% | - |
| 51+ | - | - | 20% | 40% | 30% | 10% |

**環境ボーナス**
特定条件下でレアリティが1段階アップグレードされる確率が上昇します：

- 満月の夜: +10%
- 雷雨: +10%
- ディープダーク: +15%
- エンド次元: +20%
- ネザー要塞: +10%
- 合計上限: 50%

### 3. 性格システム

テイム時に8種類からランダムに決定されます。

| 性格 | 効果 |
|------|------|
| 勇敢 | 攻撃+15%、HP-5% |
| 頑丈 | HP+20%、移動速度-10% |
| 俊敏 | 移動速度+20%、HP-10% |
| 忠実 | 親密度獲得+30% |
| 狂暴 | 攻撃+25%、被ダメージ+15% |
| 慎重 | 被ダメージ-15%、攻撃-10% |
| 幸運 | ドロップ品質+20% |
| 天才 | 経験値獲得+25% |

### 4. 親密度システム

ペットと行動することで親密度が上昇し、ボーナスが解放されます。

| レベル | 必要EXP | ボーナス |
|--------|---------|---------|
| Lv.1 | 0 | なし |
| Lv.2 | 100 | 移動速度+5% |
| Lv.3 | 300 | オーナーに再生効果 |
| Lv.4 | 600 | ステータス+10% |
| Lv.5 | 1000 | 覚醒可能 |

**親密度の獲得/減少**
- 戦闘キル: +3（5秒クールダウン）
- エサやり: +5（10秒クールダウン）
- ログイン時召喚: +1
- 日次減衰: -1/日（最大-30）
- ペット死亡: -10

### 5. 基礎ステータス

Mob種族に応じたHP/攻撃力の基礎値がランダムに振られます。

**ステータス計算式:**
```
最終値 = (基礎値 + 強化値) × レアリティ倍率 × 性格補正 + 親密度ボーナス + 装備ボーナス
```

ステータスの再抽選・+1強化はconfig指定のアイテムで可能です。

### 6. 進化システム

特定条件を満たすとペットが上位種に進化できます。

| 進化元 | 進化先 | 必要レベル | 必要親密度 | 必要アイテム |
|--------|--------|-----------|-----------|-------------|
| ゾンビ | ハスク | 30 | Lv.3 | 金のリンゴ |
| ハスク | ドラウンド | 50 | Lv.4 | 海洋の心 |
| スケルトン | ウィザースケルトン | 40 | Lv.4 | ウィザスケの頭 |
| スパイダー | 洞窟グモ | 25 | Lv.3 | 発酵したクモの目 |

- `/petevolve` で確認GUI表示 → 承認後に進化実行
- 進化時はステータスボーナスが加算
- 失敗時は自動ロールバック（データ消失なし）

### 7. 固有スキル

Epic以上のレアリティで解放される種族固有スキル。

| スキル | 対象Mob | 効果 | クールダウン |
|--------|---------|------|-------------|
| 自爆 | クリーパー | 範囲爆発ダメージ（自傷あり） | 60秒 |
| 遠吠え | ウルフ | 範囲スロー効果 | 45秒 |
| テレポート同行 | エンダーマン | オーナー位置にテレポート | 30秒 |
| 炎の雨 | ブレイズ | 範囲炎上ダメージ | 60秒 |

### 8. ペット装備

3つの装備スロットでペットを強化できます。

| スロット | 効果 |
|---------|------|
| HELMET | 防御系ステータス |
| CHEST | HP/防御ステータス |
| ACCESSORY | 特殊効果 |

- `/petequip` でGUI表示
- 装備にPDCタグ `mypetaddon:stat_<statName>` を付与することでステータスボーナス

### 9. ペット図鑑

テイムしたMob種族を記録する図鑑システム。

- `/petdex` でページネーション付きGUI表示
- テイム済み: スポーンエッグアイコン（グロウ表示）、テイム回数、最高レアリティ表示
- 未テイム: グレーアイコン（???表示）
- ソート: 名前順 / レアリティ順 / テイム回数順（config変更可）

**コンプリート報酬**

| 達成率 | 報酬 |
|--------|------|
| 25% | テイム成功率+5% |
| 50% | テイム成功率+10% |
| 75% | レアリティUP確率+5% |
| 100% | 称号「ペットマスター」 |

## コマンド一覧

| コマンド | 説明 | 権限 |
|---------|------|------|
| `/petname <名前>` | テイム中のモンスターに名前を付ける | `mypetaddon.tame` |
| `/petstatus` | ペットの詳細ステータスを表示 | `mypetaddon.status` |
| `/petskill [skill]` | 固有スキルの使用/一覧表示 | `mypetaddon.skill` |
| `/petevolve` | ペットの進化（確認GUI付き） | `mypetaddon.evolve` |
| `/petequip` | 装備管理GUI | `mypetaddon.equip` |
| `/petdex` | ペット図鑑GUI | `mypetaddon.encyclopedia` |
| `/petadmin reload` | 設定ファイル再読込 | `mypetaddon.admin` |
| `/petadmin setrarity <player> <rarity>` | レアリティ強制変更 | `mypetaddon.admin` |
| `/petadmin setbond <player> <amount>` | 親密度強制変更 | `mypetaddon.admin` |
| `/petadmin migrate` | レガシーデータ移行 | `mypetaddon.admin` |

## 設定ファイル

`config.yml` で以下の全パラメータを調整可能です：

- テイムアイテム・タイムアウト・除外リスト
- レアリティ確率テーブル・環境ボーナス・LevelledMobs連携設定
- 性格の種類・効果値・出現ウェイト
- 親密度の獲得/減少量・レベル閾値・デバウンスクールダウン
- 基礎ステータスの範囲（Mob種別ごと）
- 進化チェーン・条件・ステータスボーナス
- 固有スキルのパラメータ（ダメージ・範囲・クールダウン）
- 装備スロット構成
- 図鑑GUI設定・達成報酬
- 全メッセージテキスト（日本語対応）
- サウンド設定

## 技術仕様

### データ保存
- **SQLite** + **HikariCP** コネクションプーリング
- メインスレッドでDB I/Oを実行しない非同期設計
- `ConcurrentHashMap` インメモリキャッシュ + 2秒間隔バッチフラッシュ
- サーバー停止時に全データ同期書き込み

### 外部プラグイン連携
- **LevelledMobs**: 3層フォールバック（公式API → PDC → デフォルト値）
- **MythicMobs**: 遅延初期化でMob判定
- いずれも任意依存 — 未導入でも動作

### プロジェクト構成
```
src/main/java/com/mypetaddon/
  config/       設定管理・バリデーション
  data/         DB・リポジトリ・キャッシュ・マイグレーション
  taming/       テイムシステム
  rarity/       レアリティ（LevelledMobs連携）
  personality/  性格システム
  bond/         親密度システム
  stats/        ステータス計算パイプライン
  evolution/    進化（安全な再作成+ロールバック）
  skill/        固有スキル・クールダウン
  equipment/    装備GUI・シリアライズ
  encyclopedia/ 図鑑GUI・達成報酬
  integration/  外部プラグイン連携
  command/      コマンド管理
  util/         ユーティリティ
```

## ビルド

```bash
./gradlew jar
# -> build/libs/MyPetAddon-1.0.0-SNAPSHOT.jar
```

## レガシーデータ移行

旧スクリプト版（`mypet_legacy.js`）のYAMLデータからの移行をサポートしています。
初回起動時に自動移行されます。手動実行は `/petadmin migrate` で可能です。
