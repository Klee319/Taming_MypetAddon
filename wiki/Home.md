# MyPetAddon Wiki

MyPetAddon は、MyPet プラグインを大幅に拡張するアドオンです。テイミング、レアリティ、性格、絆、進化、装備、図鑑といった多彩な育成要素を追加し、ペットとの冒険をより深く楽しめるようにします。

---

## 主な機能

| 機能 | 概要 |
|------|------|
| [テイミング](Getting-Started) | 動物やモンスターを専用アイテムで仲間にできる |
| [レアリティ](Rarity) | COMMON〜MYTHIC の6段階。ステータス倍率やスキルスロット数が変化 |
| [性格](Personality) | 8種類の性格がランダムに付与され、ステータスや特殊効果に影響 |
| [絆システム](Bond) | ペットとの絆を深めるほどステータスボーナスが増加 |
| [ステータス](Stats) | モブ種別ごとの基礎ステータスとレベルボーナス |
| [進化](Evolution) | 条件を満たすとペットが上位モブに進化 |
| [スキルツリー](Skilltree) | テイム時に自動割り当て。汎用9種+モブ固有ツリー |
| [装備](Equipment) | HEAD/CHEST/ACCESSORY の3スロットに装備を付けてステータス強化 |
| [図鑑](Encyclopedia) | テイムしたペットを記録。コンプリート報酬あり |
| [リロールアイテム](Reroll-Items) | ステータスやスキルツリーを再抽選できる特殊アイテム |

## クイックスタート

1. サーバーに MyPetAddon の JAR を `plugins/` フォルダに配置して起動
2. 動物を仲間にしたい場合は、**金リンゴ** を手に持って動物を右クリック
3. モンスターを仲間にしたい場合は、**ダイヤモンド** を手に持ってモンスターを右クリックしてから倒す
4. テイミング成功後、`/pettame <名前>` で名前を付けると完了
5. `/petstatus` でペットの詳細ステータスを確認
6. ペットを育てて絆を深め、進化や装備でさらに強化しよう

詳しくは [はじめかた](Getting-Started) を参照してください。

## コマンド早見表

| コマンド | 説明 |
|----------|------|
| `/pettame <名前>` | テイミング中のペットに名前を付ける |
| `/petstatus` | ペットのステータスを表示 |
| `/petevolve` | ペットを進化させる |
| `/petequip` | 装備GUI を開く |
| `/petdex` | 図鑑を開く |
| `/petst` | スキルツリーのスキル一覧を表示 |
| `/petrelease` | ペットを解放する（確認GUI付き） |

詳しくは [コマンド一覧](Commands) を参照してください。
