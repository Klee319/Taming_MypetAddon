# MyPetAddon - プレイヤーズガイド

MyPetの機能を大幅に拡張するアドオンプラグイン。テイミング・レアリティ・性格・絆・進化・装備・図鑑の各システムが連動し、あなただけのペットを育てることができます。

## 1. コマンド一覧

| コマンド | エイリアス | 説明 |
|---------|-----------|------|
| `/pettame <名前>` | `/petclaim` | モンスターテイミング中にペットの名前を決定 |
| `/petstatus` | - | ペットのステータスを表示 |
| `/petevolve` | - | ペットの進化画面を開く |
| `/petequip` | - | 装備GUIを開く |

| `/petadmin setbond <player> <値>` | - | 絆設定（管理者） |

## 2. テイミング（捕獲）

モブにテイムアイテムを**右クリック**で使用するとテイミングを試行します。

### 動物テイミング

| アイテム | 成功率 | 消費 |
|---------|--------|------|
| 金のリンゴ | **5%** | する |
| エンチャント金のリンゴ | **100%** | する |

成功すると即座にMyPetとして登録されます。

### モンスターテイミング

| アイテム | 成功率 | 消費 |
|---------|--------|------|
| ダイヤモンド | **5%** | する |
| エンチャント金のリンゴ | **100%** | する |

成功すると名前入力フェーズに入ります。**60秒以内**に `/pettame <名前>` で名前を付けてください。

### テイミングの制限

- 1体のモブに対する試行上限: **5回**（超えると以降は使用不可）
- 同時に1体のモブにしかテイミングできない
- 他プレイヤーがテイミング中のモブには使えない

### テイミングできないモブ

エンダードラゴン、村人、行商人、シュルカー

MythicMobsが導入されている場合、MythicMobsのカスタムモブもテイミング対象外です。

### ペットのサイズ変更

テイミング後、一部の大型モブは自動的にサイズが縮小されます。

| モブ | サイズ |
|-----|--------|
| ウィザー | 70% |
| エルダーガーディアン | 70% |
| ガスト | 50% |

|------|-----------|------------|------------|------------|
| 1 | 0 | +0% | +0% | +0% |

| 3 | 600 | +10% | +7% | +5% |
| 4 | 1,200 | +18% | +12% | +8% |
| 5 | 2,000 | +28% | +20% | +12% |

### 絆EXPの獲得

| 行動 | 獲得量 | クールダウン |
|------|--------|-----------|
| 戦闘でモブを倒す | +1 | 10秒 |

|------|--------|

| オフライン中の累積最大減少 | -80 |

## 6. ステータスとレベル

ペットのステータスは**基礎値 x レアリティ倍率 x 性格補正 x 絆ボーナス x レベルボーナス + 装備効果**で決まります。

### モブ別基礎ステータス範囲

テイム時にティア内でランダムに決定されます。

| ティア | 体力 | 攻撃力 | 速度 | 該当モブ例 |
|--------|------|--------|------|-----------|
| Tier 1 | 10-20 | 1-3 | 0.20-0.30 | ニワトリ, ウサギ, コウモリ, シルバーフィッシュ |
| Tier 2 | 20-40 | 2-5 | 0.22-0.32 | ウシ, ウマ, オオカミ, ネコ, イルカ |
| Tier 3 | 15-30 | 4-8 | 0.25-0.35 | ゾンビ, スケルトン, クリーパー, エンダーマン |
| Tier 4 | 50-100 | 6-14 | 0.25-0.38 | ブレイズ, ガスト, ラヴェジャー, アイアンゴーレム |
| Tier 5 | 100-150 | 12-25 | 0.28-0.40 | エルダーガーディアン |
| Wither | 150-200 | 15-30 | 0.30-0.42 | ウィザー |
| Warden | 200-250 | 20-40 | 0.30-0.45 | ウォーデン |
| Wolf/Bee | 25-50 | 3-7 | 0.30-0.40 | オオカミ, ミツバチ |

### レベルボーナス

MyPetのレベルが上がることでステータスにボーナスが付きます。

- **経験値獲得倍率**: 0.7倍（通常より30%遅い）
- **ボーナス開始**: Lv20以降
- **1レベルあたり**: +0.625%
- **上限**: +50%（Lv100で到達）
- **経験値ロス防止**: 有効（死亡やログアウトでレベルが下がることはありません）

| レベル | ボーナス |
|--------|---------|
| Lv 20 | +0% |
| Lv 30 | +6.25% |
| Lv 50 | +18.75% |
| Lv 70 | +31.25% |
| Lv 100 | +50%（上限） |

## ステータス計算式

```
最終ステータス = (基礎値 x レアリティ倍率 x 性格補正 x (1 + 絆ボーナス) x (1 + レベルボーナス)) + 装備効果
```

**計算例**: ゾンビ (攻撃基礎値6) + LEGENDARY (x2.0) + 獰猛 (x1.20) + 絆Lv5 (+20%) + Lv50 (+18.75%) + ドラゴンブレス (+6)

→ 6 x 2.0 x 1.20 x 1.20 x 1.1875 + 6 = **26.5**
